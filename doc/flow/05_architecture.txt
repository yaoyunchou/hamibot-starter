架构调用流程
============

─────────────────────────────────────────
当前架构（纯本地，不依赖外部服务器）
─────────────────────────────────────────

Hamibot 脚本
    └── lib/service.ts
            │  (只请求 localhost，无凭据)
            └──► localhost:3000  (server/ FastAPI)
                        │
                        └── data/*.json  (纯本地 JSON 文件存储)


─────────────────────────────────────────
server 内部结构
─────────────────────────────────────────

server/
    ├── main.py                  ── FastAPI 入口，注册路由
    ├── core/
    │   ├── config.py            ── 从 .env 读取配置（端口）
    │   ├── local_store.py       ── 本地 JSON 文件读写工具
    │   └── device_logger.py     ── 设备日志写入本地文件
    ├── routers/
    │   ├── logs.py              ── POST /api/logs            写本地设备日志
    │   ├── order.py             ── GET  /api/order/good      按 nickName+title 查询
    │   │                           GET  /api/order/number    按 orderNumber 查询
    │   │                           POST /api/order           新增/更新订单
    │   ├── cache.py             ── GET  /api/cache/element   读元素缓存
    │   │                           PUT  /api/cache/element   写元素缓存
    │   ├── book.py              ── GET  /api/book            按 title 搜索
    │   │                           POST /api/book/view       更新曝光数据
    │   │                           GET  /api/spread          获取推广列表
    │   │                           POST /api/fsbook          新增樊登书籍
    │   │                           GET  /api/fsbooks         获取樊登书籍列表
    │   └── goods.py             ── GET  /api/goods           按 nickName+title 查询
    │                               POST /api/goods           新增/更新商品
    └── data/
        ├── element_cache.json   ── 元素缓存
        ├── orders.json          ── 订单数据
        ├── books.json           ── 书籍数据
        ├── fs_books.json        ── 樊登书籍数据
        ├── spread.json          ── 推广配置
        ├── goods.json           ── 商品数据
        ├── server.log           ── 服务器运行日志
        └── device_logs/         ── 设备日志（按 name/日期 分目录）


─────────────────────────────────────────

            └──► localhost:3000  (server/ FastAPI)
